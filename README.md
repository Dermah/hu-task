# hu-task

## Prerequesites

You'll need

- [Node](https://nodejs.org)
- [Yarn](https://yarnpkg.com/en/docs/install)

## Install

Clone the repo:

```sh
git clone https://github.com/Dermah/hu-task.git
cd hu-task
```

Then add dependencies:

```sh
yarn test
```

## Develop

Run tests:

```sh
yarn test
```

Or run tests with file watching:

```sh
yarn test --watch
```
